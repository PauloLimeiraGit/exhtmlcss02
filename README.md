Este projeto foi desenvolvido como parte do meu aprendizado em desenvolvimento web, com foco na criação de uma página de receitas culinárias utilizando HTML e CSS. O objetivo foi praticar a estruturação de conteúdos como ingredientes, modo de preparo e títulos de forma organizada e semântica, além de aplicar estilos para melhorar a leitura e a experiência do usuário. Também foram explorados conceitos como layout responsivo, tipografia e uso de cores para destacar informações importantes. A página, embora simples, é funcional e representa um passo importante na evolução para projetos mais completos e interativos.
